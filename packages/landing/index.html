Ritual Agent Wallet

MPC wallets built for AI agents on Ritual Chain. Give your AI agents secure, threshold-signed wallets they can control through HTTP APIs or MCP. No browser extensions. No exposed seed phrases. Full webhook and alert support for every critical wallet event.

Shamir 2-of-3 MPC HMAC-secured webhooks Real-time alert rules MCP compatible MIT Licensed

Connect Your Agent in Minutes

Works with Claude Desktop, Cursor, Cline, Windsurf, and VS Code. Install with a single configuration and start managing wallets immediately.

Transport Options

Pick how the MCP server runs. Hosted is recommended — no Node install required, ready in seconds. Local gives you full control on your own machine.

Instant setup. No Node.js installation required. Authenticate every request with a bearer token.

Don't have a token? Use the public demo endpoint above and ask the maintainer for a bearer token.

Run the MCP server directly on your machine using Node.js 18+ for maximum control and customization.

No service yet? Run npm install && npm run dev:service from the monorepo and use http://localhost:3000. The first run prints an admin key — that's your RITKEY_API_KEY.

Import and Export

  
Bring your own private key or export wallets to MetaMask, Rabby, or other compatible wallets. Exported wallets are automatically archived for additional safety.


  
Sybil Protection

  
One wallet per API key by default to prevent abuse and simplify wallet management.


  
Troubleshooting


  
Tools Not Appearing

  
Completely restart your MCP client after installation. MCP servers launch during application startup.


  
"npx" Not Found

  
Install Node.js 18 or newer from nodejs.org.


  
Authentication Errors

  
    
    
  • Hosted Mode: verify your bearer token.
    • 
    
  • Local Mode: verify that your RITKEY_API_KEY matches the key generated by your service.
