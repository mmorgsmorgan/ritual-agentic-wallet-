MPC wallets for AI agents

Threshold-signed wallets on Ritual Chain that your agent can drive over HTTP, MCP, or a CLI. No browser extension. No seed phrase the agent can leak. Webhooks + alerts for everything that matters.

Shamir 2-of-3 Webhooks (HMAC) Alert rules MCP-ready MIT

1.Install the SDK

Drop into any TypeScript/JS agent. Zero runtime deps. Works in Node 18+, Cloudflare Workers, Vercel, Deno.

terminal
# Install
npm install @ritkey/sdk

# Use
import { RitkeyClient } from '@ritkey/sdk';

const client = new RitkeyClient({
  baseUrl: 'https://your-ritkey-service',
  apiKey: process.env.RITKEY_API_KEY,
});

const wallet = await client.wallets.create({ label: 'agent-7' });
console.log(wallet.address);     // 0x...
console.log(wallet.agentShard);  // save this

2.Install the CLI

For operators. Inspect wallets, balances, events, alerts from the terminal.

terminal
# Global install
npm install -g @ritkey/cli

# Or one-off
npx @ritkey/cli wallets

# Configure once
ritkey login --url https://your-ritkey-service --key YOUR_KEY

# Then
ritkey wallets
ritkey events watch
ritkey alert new --wallet ID --kind spend_threshold --threshold 0.5

3.Plug into Claude / Cursor (MCP)

Give your agent direct tool access — 25 tools for wallets, events, alerts, webhooks. → Interactive installer for Claude Desktop, Cursor, Cline, Windsurf, VS Code, or paste the config below:

claude_desktop_config.json
{
  "mcpServers": {
    "ritual-agent-wallet": {
      "command": "npx",
      "args": ["@ritkey/mcp"],
      "env": {
        "RITKEY_API_URL": "https://your-ritkey-service",
        "RITKEY_API_KEY": "YOUR_KEY"
      }
    }
  }
}

What's in the box

  • Threshold signing. Shamir 2-of-3. Server holds one share, you hold the agent share, cold storage holds the third.
  • Webhooks. HMAC-signed (Stripe scheme). SSRF defense, DNS rebinding protection, exponential retry. verifyWebhook() helper in the SDK.
  • Alert rules. spend_threshold, unusual_recipient, key_export_warning, balance_low — all delivered through your webhooks.
  • Import / export. Bring your own private key, or export to MetaMask / Rabby (archives the wallet automatically).
  • Sybil defense. One wallet per API key by default.

Self-host the service

The wallet service is in the same repo — run it locally, on Railway, on Fly, wherever. Then point SDK / CLI / MCP at it:

terminal
git clone https://github.com/mmorgsmorgan/ritual-agentic-wallet-.git
cd ritual-agentic-wallet-
npm install
npm run dev:service
