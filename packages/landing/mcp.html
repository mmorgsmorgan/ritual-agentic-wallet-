Add the Ritkey MCP to your agent

25 tools for wallets, balances, events, alerts, and webhooks — over MCP. Plug into Claude Desktop, Cursor, Cline, Windsurf, or VS Code with one config.

@ritkey/mcp@0.1.0 25 tools stdio + HTTP MIT

1.Pick a transport

The Hosted MCP runs on Railway — no Node install, instant. Local spawns the server on the user's machine via npx — more control, requires Node 18+.

Don't host your own? Use the public demo: https://zooming-gentleness-production-0d17.up.railway.app/mcp — ask the maintainer for a token.
No service yet? Run npm install && npm run dev:service from the monorepo and use http://localhost:3000. The first run prints an admin key — that's your RITKEY_API_KEY.

Troubleshooting

  
    
    
  • Tools don't show up. Fully quit and restart the client — MCP servers are launched on app start.
    • 
    
  • "command not found: npx". Install Node 18+ from nodejs.org or via your package manager.
    • 
    
  • Auth errors. Hosted mode: re-check the bearer token. Local mode: confirm RITKEY_API_KEY matches the key your service printed on first run.
    • 
    
  • Pin a version. Replace @ritkey/mcp (local) or mcp-remote (hosted) with a versioned spec in the args array.
